# 504 E Black Diamond Dr - Real Estate Listing

A high-end, single-page real estate listing for the property at **504 E Black Diamond Dr, Casa Grande, AZ 85122**.

## Features
- **Modern Design:** Built with React, Tailwind CSS, and Google Fonts.
- **Solar Benefits:** Detailed section explaining the grandfathered solar rates and savings.
- **Interactive Map:** Embedded Google Map for location details.
- **Property Gallery:** High-resolution visual tour of the residence.
- **Coming Soon Bypass:** Includes a hidden bypass for authorized viewers.
